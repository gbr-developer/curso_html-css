Aprendendo a criar Media Queries!

Media Queries são diferentes versões de estilos para o mesmo site ficar adaptável à vários dispositivos

Media types: define um padrão de estilos
screen: tela
print: impressão
all: todos

Neste exemplo, foram criados uma versão de estilo para tela e outra para impressão. Clique na tela com o botão direito e depois clique em imprimir para ver o estilo de impressão.
