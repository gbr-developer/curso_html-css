Controles sem input

Controles

select: Define uma lista com opções por seleção
option: Adiciona as opções do select
optgroup: Cria um grupo para as opções

datalist: Define uma lista com opções digitadas
option: Adiciona sugestões na lista
Precisa de um input do type text e um atributo list="id_do_datalist" dentro do input

textarea: Área onde o usuário pode escrever uma mensagem
cols e rows: Definem o tamanho da área de texto com colunas e linhas
É necessário um method="post" no formulário
