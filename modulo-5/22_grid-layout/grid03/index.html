Grids implícitas

grid-auto-rows - Define automaticamente um tamanho de linha para novos elementos que entrarem no grid implicitamente

grid-auto-columns - Define automaticamente um tamanho de colunas para novos elementos que entrarem no grid implicitamente

grid-auto-flow - Define se novos elementos adicionados implicitamente no grid entrarão em uma nova linha ou em uma nova coluna

Valores para Grids

Para Grids é mais interessante utilizar valores fracionados, com fr. A soma dos fr irá dividir o container no resultado e distribuir os elementos na proporção definida para preencher o container

Também podem ser misturados outros valores na mesma especificação
