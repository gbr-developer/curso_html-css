Propriedades básicas do Container

Display: grid - Define o tipo do elemento para grid e transforma o elemento em um grid-container

grid-template-columns - Define as colunas do grid de acordo com a quantidade e os tamanhos que forem digitados

grid-template-rows - Define as linhas do grid de acordo com a quantidade e os tamanhos que forem digitados

gap - Define um espaçamento entre os itens

Propriedades de alinhamento

align-items: Alinha os items na vertical

justify-items: Alinha os items na horizontal

align-content: Alinha todo o conjunto de items na vertical

justify-content: Alinha todo o conjunto de items na horiznotal
