Seletores

Diferencia os elementos sem e com seletores

Id

h2 com id

Id é um seletor que só pode conter 1 elemento com aquele seletor

Exemplo HTML: id="nome_do_id"

Exemplo CSS: #nome_do_id { }

class

h2 com class

Class é um seletor que pode conter vários elementos com o mesmo seletor

Exemplo HTML: class="nome_da_classe"

Exemplo CSS: .nome_da_classe { }
